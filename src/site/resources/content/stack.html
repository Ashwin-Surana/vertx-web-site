The Vert.x stack is a set of components you can use to make building Vert.x application simpler.

Of course, you don't have to only use components in the Vert.x stack. You can use use Vert.x with all the usual Java libraries that you are used to.

Get started with the examples

We provide a range of examples demonstrating usage of Vert.x, including getting started with Maven and Gradle and usage of many different parts of Vert.x
